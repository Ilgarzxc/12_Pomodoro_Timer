## Pomodoro Timer
A lightweight browser-based Pomodoro timer built with HTML, CSS, and JavaScript.

## Features
- Start, pause, and reset a Pomodoro work session
- Simple and responsive user interface
- Includes timer audio alerts
- Designed for focused productivity cycles

## Usage
1. Open `index.html` in your browser.
2. Click the start button to begin the timer.
3. Pause or reset when needed.

## Project Structure
- `index.html` — page markup
- `style.css` — styling and layout
- `app.js` — timer logic and interaction
- `sounds/` — audio files for alerts

## Notes
This is a small personal productivity project for practicing web development and building a functional Pomodoro timer.
